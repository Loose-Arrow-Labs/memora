## What Happened
Agentmemory was cloned, built, and run locally for competitor evaluation. Its
local viewer demonstrated a strong first-run impression, runtime activity
visibility, timeline-style observability, session surfaces, replay affordances,
and automatic capture concepts.

## Why
The tool made clear that memory products for agentic workflows are expected to
show activity, captured context, and debugging value quickly. Memora's stronger
governance, deterministic retrieval, and legacy-code trust story remain
differentiated, but they need to become visible earlier in the product journey.

## Impact
Memora's roadmap shifted from external workflow first toward project import and
first run first. Observability, replay debugging, hybrid capture, GitHub
evidence import, and MCP readiness became must-have product surfaces instead of
later polish.

## Follow-up
Create governed draft artifacts for the revised roadmap, add an explicit
runtime evidence boundary, keep source repositories as evidence sources, and
prepare M10 for unattended stacked milestone execution.
