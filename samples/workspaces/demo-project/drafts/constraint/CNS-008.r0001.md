## Constraint
Advisory discovery may suggest candidate evidence or candidate memory, but it
must not be treated as final grounded context unless the result becomes an
approved or explicitly allowed Memora artifact.

No advisory retrieval provider may bypass import modes, lifecycle, approval,
provenance, safety filtering, or deterministic context assembly.

## Why It Exists
Hybrid retrieval is useful for legacy codebases because it can discover more
candidate material than deterministic artifact selection alone. Without a hard
boundary, that same discovery path could make inferred or probabilistic output
look like governed project truth.

## Implications
Candidate records need a source class such as evidence-derived, inferred, or
advisory. Agent readiness reporting should show where advisory discovery would
help later, while keeping current grounded context deterministic and
lifecycle-aware.
